# Travel Planner

Progetto web per gestire viaggi e luoghi da visitare.

## Funzionalità
- Creazione viaggi
- Visualizzazione lista viaggi
- Aggiunta luoghi da visitare
- Segnare luoghi come visitati
- Eliminare luoghi

## Tecnologie utilizzate
- HTML
- CSS
- JavaScript
- Node.js
- Express
- Prisma

## Descrizione
Il progetto permette di organizzare i propri viaggi e pianificare i luoghi da visitare in modo semplice e intuitivo.
